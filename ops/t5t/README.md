# ops/t5t/ — weekly Top-5 (Nvidia Way)

Each founder posts **five bullets of raw ground truth ≤7 days old** every Friday —
Messenger threads, leads, surveys, supplier calls — ending with **their hours split**
(traction % vs product %). File one markdown per founder per week:
`YYYY-WW-<founder>.md` (e.g. `2026-29-benjie.md`).

## Template

```
# T5T — <founder> — week 2026-WW

1. <raw ground truth: a real thread / lead / survey / call, ≤7 days old>
2. ...
3. ...
4. ...
5. ...

Hours split: traction __% / product __%
(Traction = ads, group posts, Messenger, surveys, quotes, installs, funnel/instrumentation
 that serves the Mission. Product = designer, build guide, BOM, prototype, docs.)
```

**Rules:** outcomes headline, outputs footnote (§6). Report surveys booked / deposits paid
/ installs signed — never "pages shipped." An interview not written up didn't happen.
No private founder one-on-ones — bad news goes to the group (§6).
